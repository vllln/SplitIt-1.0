# 💸 SplitIt — Честный чек

Простой и удобный консольный калькулятор на Python для разделения счета в компании.

## ✨ Основные функции
- **Расчет доли:** Делит чек на любое количество гостей.
- **Чаевые:** Автоматически добавляет 10%, 15% или 20% к сумме.
- **История трат:** Сохраняет каждый расчет в текстовый файл, создавая новый отчет для каждого месяца.
- **Безопасность:** Защита от случайного ввода букв вместо цифр.
- **Чистый интерфейс:** Очистка экрана после каждого расчета.

## 🚀 Как запустить
Убедитесь, что у вас установлен Python 3.
1. Скачайте файл `splitit.py`.
2. Запустите его через терминал: `python splitit.py`.
3. Для выхода введите слово **"стоп"**.
